# Zola + Vite + TS + Tailwind

## Prerequisites

- [Install Zola](https://www.getzola.org/documentation/getting-started/installation/)
- [Install Node](https://nodejs.org/en/download/)

## Usage

- Clone the repo
- Install node dependencies

```
npm i
```

- Start Dev Server

```
npm start
```

- Build for Production

```
npm run build
```
